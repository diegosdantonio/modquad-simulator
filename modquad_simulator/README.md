# ModQuad Simulator

## Requirements
* https://github.com/whoenig/crazyflie_ros
* install vispy numpy scipy transforms3d
```
sudo pip install --upgrade vispy numpy scipy transforms3d
```


## Launching the simulator
```
$ roslaunch modquad_simulator simulation.launch
```
